Sistema de Administracion de Colegio en Java

Resumen

Este proyecto consistio en conectar un entorno Java a una base de datos MySql para manejar la creacion cursos, inscripcion de alumnos y asignaciond de maestros en una escuela, usando interfaz grafica y patron de diseño DAO para manejar uniformemente las solicitudes sql y la logica de la interfaz grafica.

Funcionalidades

  • Inscripcion y Retiro de Estudiantes a una materia y un curso
  • Creacion de Cursos para pasar materias
  • Asignacion de un Maestro a cada materia

Tecnologías Utilizadas

  • Java
  • MySQL para la base de datos
  • Patron de diseño DAO
  • Jframe
